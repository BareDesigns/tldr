# SC-IM

> A curses based, vim-like spreadsheet calculator.

- Start SC-IM:

`scim {file name.csv}`

- Move Between Cells:

`h, j, k, l, arrow keys`

- Enter a string into the current cell:

`<     Right Justify`
`>     Left Justify`

- Enter a numeric constant into the current cell:

`=`

- Edit Cells:

`E     Edit string`
`e     Edit numeric value`
`|     Center the string in the current cell`
